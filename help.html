Tag 1.0 help

 


To Add Tag -

Step 1.
Choose the directory from the directory selection box.
The files will be listed in the file list box
Step 2.
From the select file type menu - select the type of file you wish to change. If it is not in the list you may enter it in the Specify box
Step 3.
From the Select Tag menu - select the tag you wish to add to the file. If it is not in the list add it in the specify box
Step 4.
Click the Add tag button - the specified tags will be added to the specified files.
I.e. On specifying html as file type and fr as tag. test.html will be changes to test_fr.html
Notes:
If the Remove tag button was clicked at this point any file of the type specified would have its tag removed if it contained one.
I.e. test_fr.html would become test.html
When specifying the file type it is not necessary to include the "." i.e. .doc - but it does not matter if you do.
When specifying a tag you do not have to specify the "_" - this will be automatically added anyway. It does not matter if this is added.
Existing files will not be overwritten.


To remove a tag -

Step 1.
Choose the directory from the directory selection box.
The files will be listed in the file list box
Step 2.
From the select file type menu - select the type of file you wish to change. If it is not in the list you may enter it in the Specify box
Step 3.
From the Select Tag menu - select the tag you wish to remove from the file. If it is not in the list add it in the specify box
Step 4.
Click the Remove tag button - the specified tags will be removed from the specified files.
I.e. On specifying html as file type and fr as tag. Test_fr.html will be changes to test.html
Notes:
If the Remove tag button was clicked at this point any file of the type specified would have its tag removed if it contained one.
I.e. test_fr.html would become test.html
When specifying the file type it is not necessary to include the "." i.e. .doc - but it does not matter if you do.
When specifying a tag you do not have to specify the "_" - this will be automatically added anyway. It does not matter if this is added.
